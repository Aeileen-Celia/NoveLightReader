EpubReaderRedo is a customized version of the **EpubReader** project, originally created by **mignaway**. This open-source application, developed with ElectronJS, is designed for reading eBooks (epub, mobi, pdf). The goal is to provide a modern and efficient reading experience with enhancements to the interface and new features.

> [!note]
> This project is based on the original work by **mignaway**. The changes in this version include improvements to the reading theme. To support the original creator, [click here](https://buymeacoffee.com/mignaway) <3

## New in Version 1.0.6
- **Dark Mode**: In addition to light and sepia themes, a dark mode is now available, offering greater comfort in low-light environments.

## Testing
To clone this project locally, you will need [Git](https://git-scm.com/) and [Node.js](https://nodejs.org/en/download/) installed on your computer.
Then use the following commands:

_(Windows)_
```bash
# Clone this repository
git clone https://github.com/a5ur4/EpubReaderRedo
# Go into the repository
cd EpubReaderRedo
# Install dependencies
npm install
# Run the app
npm start
```
